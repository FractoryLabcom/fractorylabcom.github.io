Methods Documentation

compute_Keq(graph_type)¶

Compute equivalent permeability Keq of the fracture network, using graphs From both types of graph (intersections and fractures)

compute_Qexch(graph_type)¶

compute Qexch, exchanged flow by nodes. Compute also Qexch2, from the other type of graph Qexch2 for now not available for intersection graph

compute_dQ(graph_type)¶

compute flow channeling density indicator dQ from flow exchanged by fracture Qf. Qf has been computed with fracture graph (Qexch) and with intersection graph (Qexch2), so we have two measurements dQ.

compute_discharge(graph_type='None')¶

Compute Q and set them as edges attributes. Need previous Pressure computing

compute_fractures_graph_backbone()¶

Compute the fracture backbone graph from the fracture graph Node label ‘boundary’ = list[borderId] if connected to one or several Borders, else -1

compute_graph_indicators(graph_type=None, compl_info=[])¶

compute some graph indicators

compute_intersections_graph(wij_label='topo')¶

Compute the intersections graph from the dfn Nodes are Intersections Edges are connecting Fractures Node label ‘boundary’ = list[borderId] if connected to Border, else -1

compute_intersections_graph_backbone1()¶

Compute the intersection backbone graph from the intersection graph. Graph dead ends and not connected clusters are removed. Can be time consuming because of the creation of the entire intersection graph, and of its copy

compute_intersections_graph_backbone2(wij_label='topo')¶

Compute the intersection backbone graph from the fracture backbone graph. Specificity : intersection between a fracture in backbone and a fracture not in backbone is kept. Node label ‘boundary’ = list[borderId] if connected to one or several Borders, else -1. ‘bc’ = ‘boundary’ will be used for flow

compute_intersections_graph_super_backbone(wij_label='topo')¶

Compute the intersection super backbone graph from the fracture backbone graph. Specificity : intersection between a fracture in backbone and a fracture not in backbone is removed. Node label ‘boundary’ = list[borderId] if connected to one or several Borders, else -1. ‘bc’ = ‘boundary’ will be used for flow

initialise_bc(graph_type='None', wij_label='topo')¶

Set boundary conditions ‘bc’ for a given graph type, must be a connected graph. For fracture graph : Add a boundary node on ‘borderflow’ connected to boundary fracture, the attribute ‘bc’ allows to distinguish them from the real nodes. For intersection backbone graphs : the attribute ‘bc’ is already created for intersection backbone graphs, and is the same as ‘boundary’ attribute.

solve_pressure(graph_type='None')¶

compute pressure p for each nodes of the graph. Solve the problem : A*p = rhs Input : a graph with attribute ‘bc’. If ‘bc’ = -1, the node is real, if bc = 0,1,2,3,4,5, the node is on a border
