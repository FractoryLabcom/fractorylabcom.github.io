DFNLab 
******

Introduction
============

DFNLab is a collection of modules to deal with Discrete Fracture Networks. 
These modules are written in C++ and binded in Python using pybind11_.
Mathematical operations are performed with Eigen3_.
Some visualizations are performed with VTK_.
Mesh generation is performed using BAMG_.

.. _pybind11: https://github.com/pybind/pybind11
.. _Eigen3: http://eigen.tuxfamily.org/index.php?title=Main_Page
.. _VTK: https://vtk.org/
.. _BAMG: https://www.ljll.math.upmc.fr/hecht/ftp/bamg/bamg.pdf

Contact: fractory@itasca.fr
